# Claude Traffic Light - 测试文档

## 项目信息

| 项目 | 说明 |
|------|------|
| 名称 | Claude Traffic Light 🚦 |

## 功能列表

1. 红绿灯状态显示（绿/黄/红）
2. Hook 推送 + 进程监控双通道检测
3. 拖拽定位 + 位置记忆
4. 双击切换正常/迷你模式
5. 可调透明度
6. 始终置顶切换
7. 多会话独立追踪
8. 系统托盘图标随状态变色
9. 悬停显示状态摘要

> 这只是一个测试文档，用于验证写入功能是否正常。✅
